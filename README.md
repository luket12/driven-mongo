# Driven-mongo

This will be a productivity app written entirely in JS.

* MongoDB & Mongooose NodeJS Backend
* VueJS with VueX & Vue-router front-end

## Features & Functionality

* CRUD Note
* CRUD Todo Checklist
* CRUD Habit
* Dashboard overview
* Pomodoro Timers
